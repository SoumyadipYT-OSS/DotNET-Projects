### TodoWebApp
This is simple todo web app application built on ASP .NET Core Web API (.NET 9 Version) with Angular in frontend. Integrated with MsSQL Server.
- Backend (Asp .NET Core Web API) {Server-side rendering}
- Frontend (Angular + TypeScript) {Client-side rendering}
